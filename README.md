# 🚀 DSA-DailyQuestion  
### My Journey through Data Structures and Algorithms in Java  

Welcome to my daily DSA repository! This project is a collection of various problems I solve daily to strengthen my problem-solving skills and master Java-based backend development.

---

## 📂 Project Structure  
The repository is organized into packages based on topics for easy navigation:

* **`arrays`** – Basic to Advanced array manipulations.  
* **`patterns`** – Visualizing logic through star and number patterns.  
* **`sorting`** – Implementation of Bubble, Selection, and Insertion sorts.  
* **`strings`** – String manipulation and memory management (String/StringBuilder).  
* **`two-pointers`** – Efficient solutions for search and comparison.  
* **`recursion`** – Breaking down complex problems into smaller ones.

---

## 🛠️ Tech Stack  
* **Language:** Java (JDK 17/21)  
* **Editor:** VS Code / IntelliJ IDEA  
* **Version Control:** Git & GitHub
